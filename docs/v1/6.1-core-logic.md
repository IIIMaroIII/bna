### ================================= Core logic =================================

### Регистрация и доступ
    - регистрация компании = создание Owner
    - логин:
        - Owner -> по email/password
        - Admin/Employee -> по выданным credentials

### Управление людьми и структура 
    - создание сотрудников -> Admin/Owner
    - деактивация сотрудников -> Admin/Owner
    - создание команд (Team) -> Admin/Owner
    - назначение сотрудников в команды -> Admin/Owner
        - команда может состоять из 1 сотрудника

### Проекты и доступ к проектам
    - создание проектов -> Admin/Owner
    - изменение статуса проекта (active/inactive) -> Admin/Owner
    - назначение Team на Project -> Admin/Owner

### Правило доступа сотрудника к проектам
    - сотрудник видит и может выбрать для clock-in только те проекты, которые назначенны для его Team
    - если сотрудник не назначен в Team --> он не видит проектов для clock-in

### Учет рабочего времени 
    - clock-in / clock-out -> Employee
    - выбор проектов при clock-in из доступных -> Employee
    - запрет clock-in без рабочего Wi-Fi (дальше рассмотрим для Remote позиции, пока что так)
    - запрет второй активной сессии (один человек = одна смена --> нельзя залогиниться на разных девайсах в разные проекты)
    - сохранение сессии (в дальнейшем будет использовать для отчетов -> Employee в Time History -> Admin/Owner в Отчеты по сотрудникам/отчеты по проектам/для Time Overview):
        - company_id
        - user_id
        - project_id
        - start_time
        - end_time
        - duration
        - created_at
        - updated_at

### Просмотр данных
    - Employee -> видит только свои часы
    - Admin/Owner видит:
        - часы сотрудников
        - часы по проектам (сколько всего времени ушло на Проект X)
        - active sessions (кто сейчас clocked-in)

### Что пока что не входит в Core logic для MVP
    - оплата сотрудников
    - рассчет зарплаты и прибыли
    - склад 
    - уведомления
    - экспорт (отчетов и инфо в .pdf etc)
    - мобильное приложение
    - микросервисы
    - отдельные dashboards для Team/Projects
